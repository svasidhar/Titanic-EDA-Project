# Titanic Data Cleaning and Visualization

## Project Overview

This project performs data cleaning and exploratory data analysis (EDA) on the Titanic dataset using Python.

## Tasks Performed

* Data loading and exploration
* Missing value handling
* Duplicate record checking
* Outlier detection using Box Plot
* Exploratory Data Analysis (EDA)
* Data visualization

## Visualizations

* Survival Count
* Gender vs Survival
* Passenger Class vs Survival
* Age Distribution Histogram
* Correlation Heatmap
* Fare Box Plot

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* VS Code

## Dataset

Titanic Dataset (Kaggle)
